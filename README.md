🎨 Collection Site

Веб-приложение для хранения и каталогизации семейной коллекции предметов искусства.

📖 О проекте

Collection Site — это веб-приложение, разработанное для хранения информации о произведениях искусства из частной семейной коллекции. Пользователь может просматривать экспонаты, получать подробную информацию об объектах, авторах и сериях, а также выполнять поиск по базе данных.

Проект был разработан с использованием Python и Flask в качестве backend-фреймворка, SQLAlchemy для работы с базой данных и HTML/CSS для пользовательского интерфейса.

⸻

✨ Возможности

* просмотр коллекции произведений искусства;
* страницы с подробным описанием каждого объекта;
* хранение информации об авторах и сериях;
* поиск по объектам и авторам;
* структурированная база данных;
* удобный веб-интерфейс.

🗄️ Структура базы данных

Проект использует реляционную базу данных со следующими сущностями:

Objects

* название объекта;
* автор;
* год создания;
* место создания;
* история создания;
* история попадания в коллекцию;
* материалы;
* фотографии;
* серия.

Creators

* имя автора;
* дата рождения;
* информация об авторе.

Series

* название серии;
* год создания.

⸻

📸 Интерфейс

Главная страница позволяет ознакомиться с коллекцией.

Каталог содержит карточки произведений искусства с краткой информацией.

Для каждого объекта доступна отдельная страница с:

* фотографией;
* историей создания;
* историей появления в коллекции;
* информацией об авторе;
* дополнительными сведениями.

⸻

🚀 Запуск проекта

git clone https://github.com/elizashanker/collection-site.git
cd collection-site
pip install -r requirements.txt
python app.py

После запуска приложение будет доступно по адресу:

http://127.0.0.1:5000
